该系统支持的渲染模板种类共有 **11 种**，具体如下：

| 模板代码           | 邮件场景                         | 必须变量（部分示例）                                                                               | 备注           |
| ------------------ | -------------------------------- | -------------------------------------------------------------------------------------------------- | -------------- |
| `VERIFY_CODE`      | 邮箱验证、注册验证码、登录验证码 | `app_name`, `time`, `verification_code`                                                            | 基础验证类     |
| `PASSWORD_RESET`   | 密码重置                         | `user_name`, `reset_link`, `expire_hours`, `ip`, `os_browser`                                      | 含安全信息     |
| `PASSWORD_CHANGED` | 密码修改通知                     | `user_name`, `change_time`, `ip`, `device`                                                         | 纯通知，无操作 |
| `LOGIN_ALERT`      | 新设备/异地登录                  | `user_name`, `login_time`, `location`, `ip`, `device`, `review_link`                               | 警示型         |
| `EMAIL_CHANGE`     | 修改邮箱 – 旧邮箱确认            | `user_name`, `old_email`, `new_email`, `cancel_link`                                               | 防劫持         |
| `ORDER_CONFIRM`    | 下单/支付成功                    | `user_name`, `order_id`, `order_time`, `items[]`, `total`, `pay_method`, `order_url`               | 电商/支付      |
| `SHIP_NOTIFY`      | 物流发货                         | `user_name`, `order_id`, `tracking_no`, `tracking_url`, `carrier`, `address`                       | 含物流查询     |
| `REFUND_STATUS`    | 退款进度                         | `user_name`, `order_id`, `refund_amount`, `status`, `reason`                                       | 状态变更       |
| `ACCOUNT_WELCOME`  | 注册欢迎                         | `user_name`, `verify_link`(可选), `onboarding_steps[]`                                             | 生命周期类     |
| `PROMO_BATCH`      | 促销/优惠券                      | `user_name`, `event_title`, `discount_desc`, `coupon_code`, `valid_until`, `goods[]`, `banner_url` | 营销类，需退订 |
| `NEWSLETTER`       | 周刊/资讯                        | `user_name`, `subject`, `articles[]`, `unsubscribe_url`                                            | 内容类         |

以上即为该邮件系统所支持的 **11 种渲染模板类型**，覆盖了事务类、营销类、生命周期类等常见邮件场景。
