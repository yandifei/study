### 一、基础邮件发送（smtplib + email）

Python 内置 `smtplib` 和 `email` 库，无需安装第三方包即可发送邮件。

#### 1.1 最简示例（纯文本）

```python
import smtplib
from email.mime.text import MIMEText

# 配置
SMTP_SERVER = "smtp.qq.com"
SMTP_PORT = 465
SENDER_EMAIL = "your_qq@qq.com"
AUTHORIZATION_CODE = "your_auth_code"   # QQ邮箱授权码，不是密码
RECEIVER_EMAIL = "receiver@example.com"

# 构造邮件
msg = MIMEText("您的验证码是：123456", "plain", "utf-8")
msg["From"] = SENDER_EMAIL
msg["To"] = RECEIVER_EMAIL
msg["Subject"] = "验证码"

# 发送
with smtplib.SMTP_SSL(SMTP_SERVER, SMTP_PORT) as server:
    server.login(SENDER_EMAIL, AUTHORIZATION_CODE)
    server.sendmail(SENDER_EMAIL, RECEIVER_EMAIL, msg.as_string())
```

#### 1.2 发送 HTML 邮件

```python
from email.mime.text import MIMEText

html_content = "<h1>您好</h1><p>您的验证码是：123456</p>"
msg = MIMEText(html_content, "html", "utf-8")
# 其余同上
```

### 二、邮件美化：从“能看”到“好看”

#### 2.1 为什么要特殊处理？

- 邮件客户端（尤其是 **Outlook 桌面版**）使用 **Microsoft Word 引擎**渲染 HTML，不支持 `flex`、`grid`、`position` 等现代 CSS。
- 必须使用 **表格布局（`<table>`）** + **行内样式** 才能保证兼容性。

#### 2.2 美观邮件的核心原则

| 原则 | 说明 |
|------|------|
| **表格布局** | 用 `</td>` `<td>` 搭建结构，避免 `div` |
| **行内样式** | `style="..."` 写在每个标签上，避免 `<style>` 块（部分客户端忽略） |
| **固定宽度** | 主容器宽度 600px，是邮件设计的黄金标准 |
| **优雅降级** | 渐变、阴影等高级效果在不支持时退化为纯色背景 |

#### 2.3 可直接使用的 HTML 邮件模板

以下是一个支持 **暗色主题、圆角卡片、验证码高亮** 的完整模板，兼容 Outlook：

```html
<!DOCTYPE html>
<html>
<body style="margin:0; padding:0; background:#5a4b9e;">
    <table width="100%" cellpadding="0" cellspacing="0" border="0" style="background:#5a4b9e;">
        <tr>
            <td align="center" style="padding:30px 10px;">
                <table width="600" cellpadding="0" cellspacing="0" border="0" style="max-width:600px; background:#2c1e5e; border-radius:20px; overflow:hidden;">
                    <!-- 头部 -->
                    <tr>
                        <td bgcolor="#4a3a8f" align="center" style="padding:35px 30px; background: linear-gradient(135deg, #667eea, #5a3fd1);">
                            <h1 style="color:#ffffff; font-size:26px;">{app_name}</h1>
                        </td>
                    </tr>
                    <!-- 内容：验证码 -->
                    <tr>
                        <td style="padding:45px 35px; text-align:center;">
                            <p style="color:#ffffff;">您好，您的验证码是：</p>
                            <div style="font-size:46px; font-weight:bold; color:#ffffff; background:#1f1547; padding:18px 40px; border-radius:12px;">
                                {code}
                            </div>
                            <p style="color:#d0c4ff;">有效期5分钟，请勿泄露。</p>
                        </td>
                    </tr>
                    <!-- 页脚 -->
                    <tr>
                        <td style="padding:30px; text-align:center; background:#25194f;">
                            <p style="color:#9b8fd1;">© {year} {app_name} | 系统自动发送</p>
                        </td>
                    </tr>
                </table>
            </td>
        </tr>
    </table>
</body>
</html>
```

> **使用说明**：将 `{app_name}`、`{code}`、`{year}` 替换为实际值。

### 三、在邮件中嵌入图片（PNG/Logo）

#### 3.1 三种主流方式对比

| 方式 | 原理 | Outlook兼容性 | 推荐度 |
|------|------|---------------|--------|
| **CID（Content-ID）** | 图片作为附件，通过 `cid:` 引用 | 最佳 | ⭐⭐⭐⭐⭐ |
| **Base64 内嵌** | 图片数据直接写在 HTML 中 | 一般（体积大） | ⭐⭐⭐ |
| **外部链接** | `<img src="https://...">` | 最差（常被屏蔽） | ⭐ |

#### 3.2 CID 方式实现步骤

**① 修改邮件构造方式：使用 `MIMEMultipart("related")`**

```python
from email.mime.multipart import MIMEMultipart
from email.mime.text import MIMEText
from email.mime.image import MIMEImage

msg = MIMEMultipart("related")   # 关键：related 类型
msg["From"] = SENDER_EMAIL
msg["To"] = RECEIVER_EMAIL
msg["Subject"] = "验证码"

# 附件 HTML
html = """<img src="cid:logo" alt="Logo" /> 您的验证码：123456"""
msg.attach(MIMEText(html, "html", "utf-8"))

# 附加图片
with open("logo.png", "rb") as f:
    img = MIMEImage(f.read())
    img.add_header('Content-ID', '<logo>')   # 必须和 cid:logo 匹配
    msg.attach(img)
```

**② 在 HTML 中引用**

```html
<img src="cid:logo" style="width:80px; height:80px;" />
```

> ✅ 图片会随邮件一起发送，收件人打开邮件时自动显示，无需点击“下载图片”。

### 四、解决 Outlook 特殊兼容性问题

#### 4.1 Outlook 的“雷区”

| 特性 | Outlook支持情况 | 替代方案 |
|------|----------------|----------|
| `flex` / `grid` | ❌ 不支持 | 使用 `表格` |
| `border-radius` | ❌ 不支持 | 忽略（优雅降级） |
| `background: linear-gradient` | ❌ 不支持 | 额外设置 `bgcolor` 纯色后备 |
| `position` / `float` | ❌ 不支持 | 使用 `align="center"` 等表格属性 |
| `<style>` 块 | ⚠️ 部分忽略 | 优先使用行内样式 |
| `max-width` / `width` | ⚠️ 部分支持 | 设置固定 `width` 并用表格包裹 |

#### 4.2 最佳实践清单

- ✅ 主结构使用 `<table>` + ` <td>`
- ✅ 所有样式写成 **行内样式**（如 `style="color:red;"`）
- ✅ 为背景色添加 `bgcolor` 属性作为后备
  ```html
  <td bgcolor="#4a3a8f" style="background: linear-gradient(...);">
  ```
- ✅ 图片设置固定宽高，避免使用百分比
- ✅ 避免使用 JavaScript、`iframe`、`video` 等元素
- ✅ 测试工具推荐：**[Litmus](https://www.litmus.com/email-testing)** 或 **[PutsMail](https://www.putsmail.com/)**（免费）

### 五、进阶方案：使用 MJML 自动生成兼容邮件

手写表格布局繁琐且易错，推荐使用 **MJML**（Mail Jet Markup Language）。

#### 5.1 MJML 的优势

- 编写简洁的类 XML 代码，自动编译为兼容所有客户端的 HTML
- 内置响应式支持，自动处理 Outlook 的特殊规则
- 免费开源，有在线编辑器 [MJML Online](https://mjml.io/try-it-live)

#### 5.2 一个 MJML 示例

```mjml
<mjml>
  <mj-body background-color="#5a4b9e">
    <mj-container>
      <mj-section background-color="#4a3a8f">
        <mj-column>
          <mj-image src="cid:logo" width="80px" />
          <mj-text align="center" color="#ffffff" font-size="26px">
            {app_name}
          </mj-text>
        </mj-column>
      </mj-section>
      <mj-section>
        <mj-column>
          <mj-text align="center" color="#ffffff">您的验证码</mj-text>
          <mj-text align="center" color="#ffffff" font-size="46px" font-weight="bold">
            {code}
          </mj-text>
        </mj-column>
      </mj-section>
    </mj-container>
  </mj-body>
</mjml>
```

编译后自动生成一套完整的表格布局 HTML，直接用于 Python 邮件发送。

#### 5.3 Python 中使用 MJML

```bash
pip install mjml
```

```python
import mjml

with open("template.mjml", "r", encoding="utf-8") as f:
    mjml_content = f.read()

html_output = mjml.to_html(mjml_content)
# 然后用 html_output 作为邮件正文
```

### 六、完整示例：带 Logo 的验证码邮件（Python + CID）

```python
import smtplib
from email.mime.multipart import MIMEMultipart
from email.mime.text import MIMEText
from email.mime.image import MIMEImage

# 配置
SMTP_SERVER = "smtp.qq.com"
SMTP_PORT = 465
SENDER = "your_qq@qq.com"
AUTH_CODE = "your_auth_code"
RECEIVER = "user@example.com"
CODE = "123456"
APP_NAME = "ACG鉴赏图廊"

# HTML 模板（使用表格布局 + cid:logo）
html_template = """
<!DOCTYPE html>
<html>
<body style="margin:0; padding:0;">
    <table width="100%" cellpadding="0" cellspacing="0" border="0" style="background:#5a4b9e;">
        <tr>
            <td align="center">
                <table width="600" cellpadding="0" cellspacing="0" border="0" style="background:#2c1e5e;">
                    <tr>
                        <td align="center" style="padding:30px;">
                            <img src="cid:logo" width="80" style="border-radius:50%;">
                            <h1 style="color:#fff;">{app_name}</h1>
                        </td>
                    </tr>
                    <tr>
                        <td align="center" style="padding:20px;">
                            <p style="color:#fff;">您的验证码</p>
                            <div style="font-size:40px; color:#ffda79;">{code}</div>
                        </td>
                    </tr>
                </table>
            </td>
        </tr>
    </table>
</body>
</html>
"""

html = html_template.replace("{app_name}", APP_NAME).replace("{code}", CODE)

# 构造邮件
msg = MIMEMultipart("related")
msg["From"] = SENDER
msg["To"] = RECEIVER
msg["Subject"] = f"【{APP_NAME}】验证码"

msg.attach(MIMEText(html, "html", "utf-8"))

# 附加图片
with open("logo.png", "rb") as f:
    img = MIMEImage(f.read())
    img.add_header("Content-ID", "<logo>")
    msg.attach(img)

# 发送
with smtplib.SMTP_SSL(SMTP_SERVER, SMTP_PORT) as server:
    server.login(SENDER, AUTH_CODE)
    server.sendmail(SENDER, RECEIVER, msg.as_string())

print("邮件发送成功")
```

### 七、常见问题与排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 图片不显示 | CID 不匹配 | 检查 `cid:logo` 和 `Content-ID: <logo>` 是否一致（包括尖括号） |
| Outlook 背景色丢失 | 渐变不被支持 | 添加 `bgcolor="#xxxxxx"` 属性 |
| 邮件进入垃圾箱 | 发件人信誉低或内容像广告 | 使用企业邮箱，避免过多链接和敏感词 |
| 中文乱码 | 编码问题 | 所有 MIMEText 添加 `"utf-8"` 参数 |
| 验证码未自动填充 | 邮件客户端不支持 | 提醒用户手动复制 |
