HTML基础结构

每个HTML文档都应包含以下基本结构：

<!DOCTYPE html> <html> <head> <title>页面标题</title> </head> <body> <!-- 页面内容 --> </body> </html>
  • <!DOCTYPE html> - 声明文档类型
  • <html> - 根元素
  • <head> - 包含元数据和文档信息
  • <body> - 包含可见的页面内容

常用HTML标签

  • 标题: <h1> 到 <h6>
  • 段落: <p>
  • 链接: <a href="url">链接文本</a>
  • 图像: <img src="image.jpg" alt="描述">
  • 列表: <ul>, <ol>, <li>
  • 表格: <table>, <tr>, <td>
  • 表单: <form>, <input>, <button>
<h1>这是一个标题</h1> <p>这是一个段落</p> <a href="https://example.com">这是一个链接</a>

HTML属性

属性提供有关HTML元素的附加信息：

  • id - 元素的唯一标识符
  • class - 为元素指定一个或多个类名
  • style - 内联CSS样式
  • src - 指定外部资源的路径（如图像）
  • href - 指定链接的目标URL
  • alt - 图像的替代文本
<img src="photo.jpg" alt="风景照片" class="image-style" id="main-photo"> <a href="page.html" style="color: blue;">链接</a>

HTML5新特性

  • 语义化标签: <header>, <footer>, <nav>, <article>, <section>
  • 多媒体: <audio>, <video>
  • 图形: <canvas>, <svg>
  • 表单增强: 新的input类型 (email, date, range等)
  • API: 本地存储, 地理定位, 拖放功能
<header> <nav> <ul> <li><a href="#home">首页</a></li> </ul> </nav> </header> <article> <h2>文章标题</h2> <p>文章内容...</p> </article> <footer> <p>版权信息</p> </footer>

1. 哪个标签用于创建超链接？

  • a) <link>
  • b) <a>
  • c) <href>

2. 哪个属性用于指定图像的替代文本？

  • a) title
  • b) alt
  • c) src

3. 哪个HTML5标签用于定义文档的页脚？

  • a) <bottom>
  • b) <footer>
  • c) <end>

HTML基本语法规则

根据MDN文档，HTML遵循以下基本语法规则：

1. 元素结构

HTML元素由开始标签、内容和结束标签组成：

<tagname>内容</tagname>

2. 空元素

某些元素没有内容，称为空元素或自闭合元素：

<img src="image.jpg" alt="描述"> <br> <input type="text">

3. 嵌套元素

元素可以嵌套在其他元素内部，但必须正确闭合：

<div> <p>这是一个<strong>重要</strong>的段落</p> </div>

⚠️ 错误示例：<p>这是<em>错误</p>的嵌套</em>

4. 属性语法

属性提供元素的额外信息：

<a href="https://example.com" title="示例网站" target="_blank">链接</a>
  • 属性名和值之间用等号分隔
  • 属性值通常用引号包围（单引号或双引号）
  • 多个属性用空格分隔

5. 布尔属性

某些属性只有存在与否的区别，没有值：

<input type="checkbox" checked> <button disabled>不可点击</button>

HTML文档结构最佳实践

1. 文档类型声明

<!DOCTYPE html>

2. 语言声明

<html lang="zh-CN">

3. 字符编码

<meta charset="UTF-8">

4. 视口设置（响应式设计）

2. 空元素 某些元素没有内容，称为空元素或自闭合元素： <meta name="viewport" content="width=device-width, initial-scale=1.0">
