| 序号 | 页面名称         | 文件路径                  | 说明                                                                                                 |
| ---- | ---------------- | ------------------------- | ---------------------------------------------------------------------------------------------------- |
| 1    | 登录页           | `pages/login/login`       | 未登录时显示，支持登录/注册                                                                          |
| 2    | 首页（分类导航） | `pages/home/home`         | 展示主题入口：ACG、PC图片、风景图、AI图等，点击跳转到图片鉴赏页并传递主题参数                        |
| 3    | 图片鉴赏页       | `pages/gallery/gallery`   | 根据主题（或搜索关键词）展示图片网格，支持分页，点击图片弹出大图+AI描述+标签（可内嵌弹窗或单独组件） |
| 4    | 智能搜索页       | `pages/search/search`     | 输入自然语言，展示图片结果（复用图片鉴赏页的网格组件）                                               |
| 5    | 用户设置页       | `pages/settings/settings` | 用户信息、我的收藏、退出登录等                                                                       |

> 注意：`pages/gallery/gallery` 可以同时承担“主题图片列表”和“智能搜索结果列表”的功能，但为了逻辑清晰，保留独立的 `search` 页面。

---

## 小程序 `app.json` 配置

```json
{
  "pages": [
    "pages/login/login",
    "pages/home/home",
    "pages/gallery/gallery",
    "pages/search/search",
    "pages/settings/settings"
  ],
  "tabBar": {
    "list": [
      { "pagePath": "pages/home/home", "text": "首页" },
      { "pagePath": "pages/search/search", "text": "搜索" },
      { "pagePath": "pages/settings/settings", "text": "我的" }
    ]
  }
}
```

- 启动后先判断登录态，若未登录则跳转 `login`，否则进入 `home`。
- 登录成功后返回 `home` 并刷新。

## 各页面关键交互

### 1. 登录页
- 手机号/微信授权登录
- 新用户自动注册
- 登录成功 -> 跳转 `home`

### 2. 首页（分类导航）
- 展示主题列表（可从后端获取或写死）
- 点击某个主题 -> 跳转 `gallery?topic=acg`

### 3. 图片鉴赏页
- 接收参数 `topic` 或 `query`
- 请求后端API获取图片数组
- 展示网格图片
- 点击图片 -> 弹出模态框，显示大图、AI描述、标签、可收藏
- 支持下拉分页加载

### 4. 智能搜索页
- 输入自然语言（例如“雨中樱花”）
- 请求后端搜索API，结果以网格展示
- 点击图片同样弹出模态框（复用相同的弹窗组件）

### 5. 用户设置页
- 显示头像、昵称
- 我的收藏（列表，点击进入图片鉴赏模式）
- 退出登录按钮

## 额外需要的组件（非页面）

- **图片网格组件**：在 `gallery` 和 `search` 中复用。
- **图片详情弹窗组件**：展示大图、AI描述、标签、收藏按钮。
