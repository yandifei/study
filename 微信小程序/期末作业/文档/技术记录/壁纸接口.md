后端图像接口文档：https://t.alcy.cc/docs.html

竖屏图片（使用 acg-images-mp）：aimp, fjmp, moemp, mp, tx, xhl, ysmp
非竖屏图片（使用 acg-images）：pc, ai, bd, fj, lai, moe, ys

| 短代码 | 类型名称 | 详细说明                                                                                                             |
| :----- | :------- | :------------------------------------------------------------------------------------------------------------------- |
| pc     | PC壁纸   | 适合电脑显示器等宽屏设备的壁纸。                                                                                     |
| ai     | AI生成   | 由人工智能工具（如NovelAI、Midjourney等）生成的图片。                                                                |
| aimp   | AI竖屏   | AI生成图中专门适配手机等竖屏设备的尺寸。                                                                             |
| bd     | 白底极简 | PC大小，背景纯白，画面仅有一个人物，留白极简风格。                                                                   |
| fj     | 风景     | 自然或城市风光景观类图片。                                                                                           |
| fjmp   | 风景竖屏 | 风景类图片裁剪为竖屏尺寸，适合手机壁纸。                                                                             |
| lai    | 软萌表情 | 在LINE、微信、QQ等聊天软件中流行的表情包。源自画师しめさば的原创角色困倦涂鸦（如打呼噜「すーすー」），并非官方套图。 |
| moe    | 萌图     | 日文「萌え」，指可爱、治愈的动漫风格图片。                                                                           |
| moemp  | 萌图竖屏 | 萌图类图片裁剪为竖屏尺寸。                                                                                           |
| mp     | 横屏壁纸 | 专门用于手机但要求横屏显示的二次元图片。                                                                             |
| tx     | 头像     | 正方形比例（1:1），适合用作社交平台头像的图片。                                                                      |
| xhl    | 软萌狐   | 接口本意是星空/星海，但由于抓取或标签错误，实际返回的多为软萌大头狐狸图。                                            |
| ys     | 原神横屏 | 《原神》相关二次创作图片，且为横屏比例。                                                                             |
| ysmp   | 原神竖屏 | 《原神》相关二次创作图片，且为竖屏比例，适合手机壁纸。                                                               |
