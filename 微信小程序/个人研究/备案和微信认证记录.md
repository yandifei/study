# 微信小程序备案
## 备案流程
填写主体/小程序信息 → 直到通过审核 → 工信部短信核验 → 等待管局终审 → 备案成功
我这里等待了4天时间，5月28日5:02分通过了审核

**验证码有效期为24小时，超时未核验或验证码输入错误累计达5次，备案申请将会被驳回，你需要重新提交备案申请，从头走一遍流程**

## 具体实践
我是5.24凌晨2点提交申请的备案
然后是10:15打电话给我说我有5个备案问题
我是个人备案的
1. 地区不够详细，我写的是学校，但是他要路口以及具体到宿舍位置
2. 说我的是英文名所以需要备注该英文的含义，并且我询问过后对方说如果是中文的话就不需要备注含义了
3. “互联网信息服务承诺书”这个需要打印出来然后手写签名，不能直接把签名图片P上去（估计是我没有作任何处理），对方说大多数都是打印后手写拍照搞的

第二次我补充了小程序的备注就给我通过了
然后11左右给我通过了审核，以此同时11：16分工信部【工业和信息化部】发短信来了要我进行验证。
去到他给的网址后输入验证码、手机号和身份证后6位就够了。后面就是等待1-20天的审核期了。


# 微信认证
**这玩意要钱，1年30元**
这个东西的作用只有1个，就是让别人能搜索到你的微信小程序，就跟你申请域名一样（但是没有域名可以用公网IP，小程序必须收钱），不过我觉得还是真的sb，我能做内网穿透和流量代理转发，也有公网IP的服务器，你这不就是硬收费吗？规定收费是你的态度，没有提供额外的方法也是你的态度。
最后给完钱还要1-3天审核时间，我服了。

5. 支付费用
