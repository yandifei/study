对于官方的docker是用于docker的，不适合Windows使用，我这里肯定是挂载数据到Windows系统的D盘，但是默认挂载到WSL的目录下（C盘），所以需要修改配置文件
```yaml
    volumes:
      - ./init:/docker-entrypoint-initdb.d
      - ./db:/data/db
      - ./configdb:/data/configdb
      - ./mongot:/data/mongot
```
还有一个就是容器启动后名字不是mongodb，所以要指定名字
```yaml
    # 容器名字
    container_name: mongodb
```

查看mongodb shell是否已经安装
```yaml
docker exec -it mongodb mongosh --version
```
