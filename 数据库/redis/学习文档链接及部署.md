# docker部署
先去`redis.conf`里把密码改了并保存，然后直接`docker compose up -d`即可

# 连接
第一种方式：
```bash
docker exec -it redis redis-cli -a 密码
```

第二种方式：
```bash
docker exec -it redis
redis-cli
auth 密码
```
