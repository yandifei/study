# 视频总结
1. 流浪猫我不该投喂，会破坏原有的生态链，他们的捕食能力会下降，放弃捕食去翻垃圾桶或者是翻鸟蛋，仅仅意识倒恐惧生存的时候才会发动捕食的本能，但是这是针对大多数猫来说的。
2. 宠物猫和流浪猫完全不同，必须区别看待，绝对不能认为猫猫可爱就以同情心去关心流浪猫，这他妈就真的是纯sb，真正会捕猎的猫被投喂捕食能力会下降，最终如果投喂了基本等死。
3. 君子论迹不论心，投喂的前提的是对猫施加一个管理，防止更多流浪猫的出现，避免投喂导致帮倒忙，最好还是TNR方法，不过想要效果好还得是跨区域和高强度的TNR。
4. 城市的流浪猫更多的是人的问题，而不是猫的问题，在城市中无论是何种动物最好的做法还是自己进行一个收容管理，而不是定期投喂
5. 定期定点投喂会被认为你是猫的饲主或者管理员，一旦你的投喂行为从“偶尔善举”变成“长期、固定的习惯”，并客观上导致了流浪动物在特定区域的聚集，你就可能被法院认定为事实上的管理人，从而需要承担相应责任，这个是写在了民法典上面的。
6. 投喂者更多出于爱心，这是一个伦理问题，难以评价，但是这些行为客观事实上给别人来了实际的损失，这是一个必须直视的问题。
