## 📚 半导体存储器知识点整理

### 一、U盘与固态硬盘（SSD）的存储本质

- **U盘和SSD属于NAND型闪存**，是半导体存储器的一种。
- **非易失性**：断电后数据不会立即丢失，可长期保存。
- **遇水丢失数据的原因**：
  - 并非电荷泄漏，而是**物理损坏**（短路、电化学腐蚀）。
  - **通电时进水**：瞬间短路烧毁主控等元件。
  - **断电时进水**：水中杂质残留会导致腐蚀，同样可能造成永久损坏。
- **进水后正确处理**：
  1. 立即断电。
  2. 吸干表面水分，自然风干48小时以上（或使用干燥剂）。
  3. 切勿通电检测或高温烘烤。
  4. 重要数据应寻求专业恢复服务。

### 二、固态硬盘的“10年寿命”概念澄清

- **两种不同含义**：
  1. **硬件使用寿命**：正常通电使用下，SSD能稳定工作约10年（取决于TBW写入总量）。
  2. **断电数据保留期限**：全新SSD断电存放时，数据可靠保存的理论最长时间。行业标准（JEDEC）规定消费级SSD仅**保证1年**，理想条件下可达10年以上。
- **通电能否延续寿命？**
  - **能延续数据保留期限**：通电后主控会自动刷新数据，防止电荷泄漏。
  - **不能逆转硬件寿命**：SSD的擦写次数（P/E Cycles，即总写入字节数TBW）是物理磨损，每次写入（包括刷新操作）都会消耗，无法靠通电恢复。
- **建议**：每隔3-6个月通电一次，让主控刷新数据。

### 三、NAND闪存颗粒的类型（SLC / MLC / TLC / QLC）

| 类型    | 全称              | 每单元存储bit | 擦写寿命（P/E Cycles） | 市场定位                 |
| :------ | :---------------- | :------------ | :--------------------- | :----------------------- |
| **SLC** | Single-Level Cell | 1 bit         | 约 10万次              | 企业级、军工             |
| **MLC** | Multi-Level Cell  | 2 bit         | 3000-1万次             | 高端企业级               |
| **TLC** | Triple-Level Cell | 3 bit         | 1000-3000次            | **当前消费级主流**       |
| **QLC** | Quad-Level Cell   | 4 bit         | 约 1000次              | 大容量消费级、冷数据存储 |

- **选购建议**：
  - 日常使用：**TLC**是最佳平衡（性能、寿命、价格）。
  - 大容量仓库盘：**QLC**性价比高，理论寿命对普通用户足够（如1TB QLC每日写入200GB，可用约13年）。

### 四、NAND与NOR闪存的区别

- **NAND闪存**：
  - 高密度、低成本，适合大容量存储（SSD、U盘、存储卡）。
  - 读取需按“页”进行，写入前必须先擦除整个“块”。
- **NOR闪存**：
  - 支持按字节随机读取，可**芯片内执行**（CPU直接运行其中代码）。
  - 容量小（几MB~几百MB），成本高，常用于BIOS、路由器固件。
- **相同点**：都是非易失性闪存。

### 五、内存（DRAM）与NOR、NAND的对比

| 对比项       | **内存（DRAM）**     | **NAND闪存**     | **NOR闪存**            |
| :----------- | :------------------- | :--------------- | :--------------------- |
| **易失性**   | **易失**（断电即丢） | 非易失           | 非易失                 |
| **主要职责** | 系统运行的“工作台”   | 海量数据的“仓库” | 核心指令的“保险箱”     |
| **速度**     | 极快（纳秒级）       | 较慢（微秒级）   | 快（支持随机字节读取） |
| **容量**     | 适中（8-64GB）       | 极大（数TB）     | 极小（几MB~几百MB）    |
| **成本**     | 较高                 | 较低             | 最高（每MB计）         |

- **一句话总结**：
  - **DRAM**：电脑运行时的临时工作空间，断电失忆。
  - **NAND**：长期存储大量文件的仓库。
  - **NOR**：存放开机等关键代码的小保险箱。

### 六、核心概念速查

- **半导体存储器**：包括DRAM、NAND、NOR等。
- **NAND闪存颗粒**：U盘/SSD的核心部件，名称来源于“NOT AND”逻辑门。
- **电荷泄漏**：NAND闪存数据丢失的理论原因（长期不通电），但进水导致的丢失是物理损坏。
- **TBW（总写入字节数）**：衡量SSD硬件寿命的关键指标，不可逆。
- **TRIM指令**：帮助SSD提前清理无效数据，维持性能。
