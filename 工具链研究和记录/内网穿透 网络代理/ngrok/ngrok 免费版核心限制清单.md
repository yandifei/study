### ngrok 免费版核心限制清单

| 限制维度           | 免费版（Free Plan）限制额度 | 避坑提示 / 说明                                                         |
| ------------------ | --------------------------- | ----------------------------------------------------------------------- |
| **月度出站流量**   | **1 GB** / 月               | 免费版最大的硬伤。传大文件或高频测试很容易用完。                        |
| **月度 HTTP 请求** | **20,000 次** / 月          | 包含所有通过隧道的 HTTP/HTTPS 请求总量。                                |
| **月度 TCP 连接**  | **5,000 次** / 月           | 仅针对 TCP 协议的连接总次数。                                           |
| **同时在线端点**   | 最多 **3 个** (Endpoints)   | 允许同时开启的公开穿透服务（隧道）数量。                                |
| **并发 Agent 数**  | 最多 **3 个** (Agents)      | 允许同时运行并连接的 ngrok 客户端实例数。                               |
| **HTTP 请求速率**  | **4,000 次** / 分钟         | 应对突发高并发流量的每分钟频率限制。                                    |
| **TCP 连接速率**   | **100 次** / 分钟           | 非 HTTP 的 TCP 流量每分钟速率限制。                                     |
| **独立域名支持**   | **1 个** 自动分配的开发域名 | 格式通常为 `xxxx.ngrok-free.app`，**不支持**绑定自己的独立域名。        |
| **网页安全警告**   | **有** (Interstitial Page)  | 浏览器首次访问时会弹出官方风险提示，访客必须手动点击确认才能进入。      |
| **协议支持**       | 支持 HTTP/S、TCP            | **不支持** 原生 TLS 隧道和任何 **UDP** 协议（无法用于大部分游戏联机）。 |
| **Webhook 验证**   | **500 次** / 月             | 使用 ngrok 自带的 Webhook 签名校验功能时的额度。                        |

---

> 💡 **关于“2小时自动断开”的误区澄清：**
> 很多早期的老教程会提到“免费版每 2 小时会自动断开并随机更换 URL”。这里帮你纠正一下：只要你**注册了账号并在本地执行了 `authtoken` 绑定**，你的免费版隧道是可以一直保持在线的，不会被强制掐断。2 小时的硬性时间限制现在只针对**完全不登录账号的匿名临时隧道**。

如果你只是偶尔本地调试一下 Webhook、写写微信小程序 Demo，免费版完全够用。但如果需要搭建长期稳定的服务，1GB 的月流量额度确实非常紧巴。
