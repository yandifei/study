# natapp介绍
Natapp 是一款专注于国内网络环境的内网穿透工具，由成都三百牛信息技术有限公司于 2015 年推出，针对国内网络延迟高、不稳定等问题做了深度优化。它支持 HTTP、HTTPS、TCP、UDP 协议，覆盖 Windows、macOS 和 Linux 系统。隧道采用 TLS v1.3 加密，并提供访问令牌、IP 黑白名单等安全控制，VIP 隧道承诺 99.9% 可用率。国内多节点 BGP 机房智能调度，配合 64 位优化客户端能提升吞吐量、降低延迟。一键开启 HTTPS 无需自行申请证书，极大方便微信小程序等开发。主要场景包括本地 API 调试、微信支付回调、项目演示、远程运维内网服务器或数据库，以及将 Ollama 等本地 AI 服务暴露到外网。使用很简单：注册购买隧道获取 authtoken，在客户端目录创建 config.ini 文件写入 authtoken=你的令牌，启动后看到 Tunnel Status Online 即连接成功；也可直接执行 ./natapp -authtoken=你的令牌。免费隧道提供随机域名，会不定期强制更换；付费 VIP 从每月 9 元起，可拥有固定子域名、绑定自主域名，享受更高带宽和稳定性。长期使用建议绑定自己的域名，通过 CNAME 记录指向 Natapp 服务器，并配置访问令牌或 IP 白名单增强安全性。凭借稳定低延迟和简单易用，Natapp 是当下国内开发者首选的内网穿透方案之一。
