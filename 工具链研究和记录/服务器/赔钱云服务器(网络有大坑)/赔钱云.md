# 🛒 低价 NAT 云服务器（VPS）完全购买指南及避坑手册

## 一、 基础参数名词解释（让你看懂图里在卖什么）
*   **CPU（核心与主频）**：核心数（如 2核/8核）决定同时处理任务的能力；主频（如 3.7GHz / 6.0GHz）决定单个任务处理的速度。如 `i9-14900K` 属于极高主频，适合吃单核性能的软件。
*   **内存（RAM）**：如 4G / 64G。越大越好，决定了能同时运行多少程序及数据库的并发能力。
*   **硬盘（系统盘 / 数据盘）**：系统盘（30G）用来装操作系统；**数据盘（如 0G / 30G / 100G）**用来装网站代码、图片、数据库等数据。
*   **带宽（下行 / 上行）**：
    *   *下行 (100Mbps)*：代表服务器下载文件的速度，理论上很快（约12.5MB/s）。
    *   *上行 (3Mbps - 45Mbps)*：代表服务器将数据传出去的速度（如用户访问你的网站）。**注意：国内低价 NAT 机上行通常会被大幅限制。**
*   **虚拟化（KVM / Hyper-V）**：
    *   *KVM*：最主流的真虚拟化技术，兼容性最好，Linux/Windows都适用，**普通用户首选 KVM**。
    *   *Hyper-V*：微软技术。**如果你打算重装 Windows 系统，Hyper-V体验极好**；但如果装老版本 Linux，可能会出现时钟不准确、网卡不识别等小Bug。
*   **NAT转发（50个）**：因为机器**没有任何独立公网IPv4**，所以商家给了你50个端口（如 20001~20050），你需要把外部端口映射到你的服务端口上才能连上服务器。
*   **售后QQ群**：这种小厂没有传统工单系统，遇到网络中断等问题必须加群找群主解决。

## 二、 ⚠️ 避坑指南：购买前必须知道的“潜在风险”与陷阱

### 🚨 1. 最大的坑：没有独立公网 IP（它是 NAT 机！）
*   **建站死穴**：国内机房**严禁解封 80 和 443 端口**。你无法用域名正常访问这台机器上的网站，访问者必须输入 `公网IP:端口号` 这种形式。
*   **严禁用途**：商家红字明确警告，**严禁用于翻墙（VPN/梯子）、端口扫描、网络攻击（DDOS）、代理发包**。一旦发现，直接封死机器且不退款，请绝对不要碰这些红线。

### 🗑️ 2. 硬盘容量骗局：警惕“0G 数据盘”！
*   **🔴 高危对象**：结合前面的表格，**`序号 1, 3, 6, 9, 12` 这几款**，虽然显示系统盘有30G，但**数据盘是 0G**。
*   **实际后果**：30G装完 Windows 或 Debian 系统后，实际留给你的剩余空间可能**只有 22G ~ 25G**。只要安装宝塔面板、拉几个 Docker 镜像，系统盘就会直接爆满。**如果你需要存放数据，绝对不要买 0G 数据盘的机型。**

### 💸 3. 跑路风险与退款陷阱
*   **跑路高危**：5元、10元一个月的机器，硬件却给到几十G内存，这严重不符合硬件成本。**这种“神仙配置”极大概率商家赚够钱就会跑路，或者节点停用。**
*   **购买策略**：强烈建议**只按“月”或“季”购买，千万不要买 2年、3年、5年（图上居然有这个选项，绝对是坑）**。
*   **特价机不退**：图中 **`序号 2 的 11元/季 618款`** 明确红字标注 **“不可退款”**，买这种机器跟抽盲盒一样，一旦网络差或连不上，你的钱就打水漂了。

### 🐌 4. 上行带宽的假象
你看到页面上写着“下行宽带 100Mbps”（这没假），但请务必看清真实的**上行带宽**（如 3Mbps 或 5Mbps）。
*   **影响**：如果上行只有 3Mbps，意味着你的服务器给外界传东西的速度只有 **300KB~400KB/s**。如果你打算用它来备份几十G的数据，或者做视频推流，那会卡到崩溃。

### 💻 5. 虚拟化与操作系统的隐藏 Bug
*   如果你买的带有 `i9-14900K` 和 `R9-9950X` 后缀的高配高频机器，它们全都是 **Hyper-V 虚拟化**。
*   如果你打算在这些 Hyper-V 机器上安装 Linux 操作系统，**强烈建议选择最新的 `Debian 12` 或 `Ubuntu 22.04` 或更新版本**。装旧版本（如 CentOS 7 或 Debian 9）可能会导致系统内网卡断流或时间变慢，极其折腾。
